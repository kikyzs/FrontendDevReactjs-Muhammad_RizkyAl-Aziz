# FrontendDevReactjs-MuhammadRizkyAl-Aziz

## Deskripsi Project
Project ini adalah implementasi Front End Developer menggunakan **Next.js 16.1.4**, **React 18+**, dan **Tailwind CSS**.  
Fitur utama:

- Tampilan daftar restoran menggunakan **grid 4 kolom** responsif
- Filter navigation:
  - Open Now (client-side filter)
  - Price (client-side filter)
  - Categories / Cuisines (server-side search)
- Detail page restaurant dengan review card
- Tombol “Learn More” tampil sebagai button **biru navy**, **teks putih**, **tanpa rounded**
- UI clean dan profesional

## Teknologi yang Digunakan
